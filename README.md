# How to install:
 - Run the requirements.txt to import all libs (pip install -r /path/to/requirements.txt)
 - Run the main.py
 - Add your Stonify Session Token
 - Click on "Start". The app refreshes after 5 min, but you can refresh manually

# .ENV config:
- CALENDAR_ID = your calendar ID (you can find on DevTools > Network > Execute a calendar request and check the calendarConfiguration payload
